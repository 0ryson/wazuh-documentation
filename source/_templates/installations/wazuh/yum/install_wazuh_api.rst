#. Add NodeJS repository:

    .. code-block:: console

      # curl -sL https://rpm.nodesource.com/setup_8.x | bash -

#. Install NodeJS:

    .. code-block:: console

      # yum install nodejs

#. Install the Wazuh API. It will update NodeJS if necessary:

    .. code-block:: console

      # yum install wazuh-api
