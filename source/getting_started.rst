Getting started guide
=====================

Wazuh is born as an OSSEC HIDS fork that quickly evolves into a more comprehensive solution for log analysis, policy monitoring, intrusion detection and policy monitoring. Some of the most important capabilities are provided based on the integration with other Open Source projects like OpenSCAP and Elastic Stack. Here is a brief description of these tools and what they do:

.. image:: images/ossec_openscap_elastic.png
   :align: center
   :width: 100%

OSSEC HIDS is the most widely used Host based Intrusion Detection System in the world, used both to get security visibility and for compliance purposes.
